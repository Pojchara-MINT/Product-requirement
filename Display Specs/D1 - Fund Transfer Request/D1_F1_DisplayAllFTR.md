# D1 — F1: Display All Fund Transfer Requests

> **Display:** D1 | **Function:** 1 of 5
> **Back to overview:** [D1_Overview.md](D1_Overview.md)

---

**Objective:** แสดงรายการคำขอโอนเงิน (FTR) ในรูปแบบ 3-Panel Layout เพื่อให้ User เห็นรายการ เอกสาร และรายละเอียดของรายการที่เลือกพร้อมกัน

> **Display Condition:** Panel 1 แสดงเฉพาะ FTR ที่อยู่ในสถานะ `PENDING_MAKER` เท่านั้น
> FTR ที่ถูก Approve (`FTR_APPROVED`) หรือ Reject (`FTR_REJECTED`) จะ**ไม่ปรากฏ**ใน List นี้

---

## Panel 1 — FTR Card List (Left)

แสดงรายการ FTR ในรูปแบบ Card เรียงจากใหม่ไปเก่า User สามารถคลิก Card เพื่อดูรายละเอียดใน Panel 2 และ Panel 3

**Header แสดง:** รายการรอโอนเงิน (N รายการ) + Checkbox เลือกทั้งหมด

**แต่ละ Card แสดง:**

| Field | Description |
|-------|-------------|
| เลขที่อ้างอิง | Reference number ของ FTR |
| Badge ประเภทรายการ | `Manual` (แดง) หรือ `System` (น้ำเงิน) |
| จำนวนเงินสุทธิ | ยอดโอน (THB) |
| ประเภทงาน | ประเภทของรายการ |
| วันที่ / เวลา | วันและเวลาที่สร้างรายการ |
| Checkbox | สำหรับเลือกรายการเพื่อดำเนินการพร้อมกัน |

**Footer (ด้านล่าง Panel 1):**

| Element | Description |
|---------|-------------|
| จำนวนรายการ | แสดงจำนวน FTR ที่ถูกเลือก |
| จำนวนเงินสุทธิรวม | ยอดรวมของรายการที่เลือก (THB) |
| ปุ่ม ยกเลิกรายการ | (แดง) ยกเลิก FTR ที่เลือก |
| ปุ่ม ยืนยันสร้างรายการ | (เขียว) Approve FTR ที่เลือก |

---

## Panel 2 — Document Viewer (Middle)

แสดงเอกสารที่แนบมากับ FTR ที่ถูกเลือกใน Panel 1

**Document Tabs:**

| Tab | Description |
|-----|-------------|
| สมุดบัญชี | รูปสมุดบัญชีของผู้รับโอน |
| บัตรประชาชน | สำเนาบัตรประชาชน |
| ใบเปลี่ยนชื่อ-นามสกุล | (ถ้ามี) |
| ทะเบียนสมรส | (ถ้ามี) |
| แบบสัมภาษณ์ | (ถ้ามี) |
| รูปอื่นๆ | เอกสารอื่นๆ ที่แนบเพิ่มเติม |

---

## Panel 3 — FTR Detail (Right)

แสดงข้อมูลรายละเอียดของ FTR ที่ถูกเลือกใน Panel 1

| Field | Description |
|-------|-------------|
| เลขที่อ้างอิง | Reference number + Badge ประเภทรายการ (Manual/System) |
| ประเภทงาน | ประเภทของรายการ |
| รหัสรายการ | รหัสรายการที่ระบบสร้าง |
| เลขบัตรประชาชน | เลขบัตรประชาชนของผู้รับโอน |
| ประเภทสินเชื่อ | ประเภทสินเชื่อที่เกี่ยวข้อง |
| หลักประกัน | ข้อมูลหลักประกัน |
| ชื่อ-สกุล (ภาษาไทย) | ชื่อผู้รับโอนภาษาไทย |
| ชื่อ-สกุล (ภาษาอังกฤษ) | ชื่อผู้รับโอนภาษาอังกฤษ |
| ช่องทาง | ธนาคารปลายทาง |
| เลขที่บัญชี / พร้อมเพย์ | หมายเลขบัญชีหรือ Proxy |
| จำนวนเงินสุทธิ | ยอดโอน (THB) |
