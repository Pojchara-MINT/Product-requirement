# D4 — F1: Transaction History List

> **Display:** D4 | **Function:** 1 of 2
> **Back to overview:** [D4_Overview.md](D4_Overview.md)

---

**Objective:** แสดงรายการ FTT ทั้งหมดในรูปแบบ Table พร้อม Filter — ให้ User ติดตามสถานะรายการ

---

## Page Layout

**Header:** สถานะงาน

| Element | Description |
|---------|-------------|
| กรองข้อมูล | ปุ่ม Filter (ซ้ายบน) — เปิด Filter Panel |
| Legend | System (จุดน้ำเงิน) / Manual (จุดชมพู) — แสดงมุมขวาบน |
| Table | แสดงรายการ FTT |

---

## Default Display

> แสดง FTT ที่มี **updated_at = วันปัจจุบัน** (Today) โดย Default
> User สามารถเปลี่ยนช่วงวันที่ได้ผ่าน Filter

---

## Table Columns

| # | Column | Description | Format |
|---|--------|-------------|--------|
| 1 | เลขที่อ้างอิง | Reference number + จุดสี (System=น้ำเงิน, Manual=ชมพู) | Text |
| 2 | ประเภท | ประเภทสินเชื่อ (ทั่วไป กู้เงินเพิ่ม / Top up / การเบิกเงิน / เปลี่ยนแปลงสัญญา / On Top / ฯลฯ) | Text |
| 3 | ช่องทางคำขอ | ช่องทางที่ยื่นคำขอมา | Online / Branch / — |
| 4 | ประเภทงาน | สินเชื่อ / คืนเงิน | Text |
| 5 | ชื่อ-สกุล | ชื่อผู้รับโอน | Text |
| 6 | ช่องทางโอนเงิน | ช่องทางที่ใช้โอนเงิน | ธนาคาร / พร้อมเพย์ |
| 7 | เลขที่บัญชี/พร้อมเพย์ | บัญชีปลายทาง | Text |
| 8 | จำนวนเงิน | ยอดโอน | ตัวเลข (บาท) |
| 9 | ผลตรวจสอบบัญชีปลายทาง | ผลการ Verify บัญชี | Badge: ผ่าน (เขียว) / ไม่ผ่าน (แดง) / ยังไม่ได้รับผล (เทา) |
| 10 | ผลเงื่อนไขโอนเงิน | ผล Internal Rule | Badge: ผ่าน (เขียว) / ไม่ผ่าน (แดง) / ยังไม่ได้รับผล (เทา) |
| 11 | สถานะรายการ | สถานะปัจจุบันของ FTT | Badge: สำเร็จ (เขียว) / ปฏิเสธ (แดง) / ยกเลิก (แดง) / รอดำเนินการ (เหลือง) |
| 12 | ผลการโอนเงินจากธนาคาร | ผลตอบกลับจากธนาคาร | Badge: Success (เขียว) / Fail (แดง) / Pending (เหลือง) |
| 13 | วันเวลาที่รายการล่าสุด | Timestamp ล่าสุดที่ FTT ถูก update | DD MMM YY \| HH:MM น. |
| 14 | วันเวลาที่รับผลและบันทึกจากธนาคาร | Timestamp ที่ได้รับผลตอบกลับจากธนาคาร | DD MMM YY \| HH:MM น. |

> Column 6 (ช่องทาง) และ Column 3 (ช่องทาง) ใช้ชื่อเดียวกันแต่ต่างความหมาย — Dev ควรแยก key ให้ชัด

---

## Filter

### Default Filter

| Filter | Default Value |
|--------|--------------|
| วันเวลาที่รายการล่าสุด (updated_at) | วันปัจจุบัน 00:00 – 23:59 |

### Filter Fields

| # | Field | Type | Default | Options |
|---|-------|------|---------|---------|
| 1 | ประเภทการค้นหา | Dropdown (Single) | ทั้งหมด | ทั้งหมด / เลขที่ลูกค้า / ชื่อ-สกุล |
| 2 | ค่าค้นหา | Text Input | — (disabled) | Enable เมื่อ Field 1 ≠ ทั้งหมด |
| 3 | วันเวลาที่รายการล่าสุด | Date + Time Range | Today 00:00 – 23:59 | User เลือกช่วงวันที่ได้ |
| 4 | ผลตรวจสอบบัญชีปลายทาง | **Multi-select** | ทั้งหมด | ผ่าน / ไม่ผ่าน / ยังไม่ได้รับผล |
| 5 | ผลเงื่อนไขการโอนเงิน | **Multi-select** | ทั้งหมด | ผ่าน / ไม่ผ่าน / ยังไม่ได้รับผล |
| 6 | สถานะรายการ | **Multi-select** | ทั้งหมด | สำเร็จ / ปฏิเสธ / ยกเลิก / รอดำเนินการ |
| 7 | ผลการโอนเงินจากธนาคาร | **Multi-select** | ทั้งหมด | Success / Fail / Pending |

**Action Button:** ปุ่ม **ค้นหา** — Apply filters
